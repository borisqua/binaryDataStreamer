Epoching

Input EEG:				sourceEEG.csv

Incoming stimulus:		"image_ID = 1" at first sample of input EEG file

Epoch duration:			1 sec after stimulus (250 samples)
						epochDur = 1 sec
						epochLen = epochDur * samplingRate = 250 samples


Output epochs:

epoch_FZ_1.csv
epoch_CZ_1.csv


Each epoch file contains 250*1 values



Output file (if option "saveEEG = true"):

eeg_20170627_152501.csv
